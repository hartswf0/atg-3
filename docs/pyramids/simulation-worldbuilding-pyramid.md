## LEVEL 1: PRINCIPLE (The worldbuilding insight)

**The Director as System Architect:**
The artist shifts from author of fixed outcomes to architect of systems — a "worlder" who sets initial conditions and allows the work to play out its own destiny.

**Bounded instantiation principle:**
- Content can be infinite
- Structure must be fixed
- Agents operate within constraints
- Emergence happens within bounds

---

## LEVEL 2: SUCCESS (What works)

### Life After BOB (Ian Cheng)
**Structure:**
- Unity engine as substrate
- Defined agent behaviors
- Open-ended emergence
- Infinite duration possible

**Why it works:**
- System is bounded (engine constraints)
- Agents are bounded (behavior rules)
- Content is emergent (unbounded)
- No cumulative state collapse

### 1000 Lives (Shuruq Tramontini)
**Structure:**
- AI turtle survival scenarios
- Fixed mechanics (mortality, learning)
- Variable instantiation (each life unique)

**Why it works:**
- Bounded failure modes
- Narrative pathos from constraint
- Each run complete unto itself

### LDraw (Community Standard)
**Structure:**
- Fixed brick definitions
- Open-ended creations
- 30-year survival

**Why it works:**
- Geometry bounded
- Expression unbounded
- Community can extend within structure

---

## LEVEL 3: FAILURE (What collapses)

### TILTH (Scene Management)
**Structure:**
- World inside world
- Cumulative narrative building
- Interconnected scenario tools

**Why it collapsed:**
- No structural bounds
- Each addition increased complexity
- Context grew, coherence degraded
- "Trough of quality" — almost good, never finished

### Modulex (Market Rejection)
**Structure:**
- Perfect 1:1 geometric cube
- Professional architectural tool
- 5mm precision

**Why it failed:**
- Geometric purity rejected by market
- Too constrained for play
- Too precise for adoption
- Structure mismatched use case

### Extended ChatGPT Sessions
**Structure:**
- Cumulative message history
- Unbounded context growth
- Scene management by default

**Why they collapse:**
- Coherence degrades at 15-20 exchanges
- Complexity grey emerges
- "Broken" and "designed" indistinguishable

## LEVEL 4: ARCHITECTURE (How to build)

### The Cheng Model
1. Choose substrate (Unity, Unreal)
2. Define agent primitives
3. Set behavior constraints
4. Allow emergent narrative
5. Never specify outcomes

### The Void Management Model
1. Choose structure (9×9 grid)
2. Define operators (Tetrad)
3. Set position constraints (81 cells)
4. Allow content variation
5. Preserve inspectability

### Key Architectural Principles
| Principle | Scene (fails) | Void (succeeds) |
|-----------|---------------|-----------------|
| State | Cumulative | Fixed |
| Content | Pre-specified | Emergent |
| Bounds | None | Structural |
| Duration | Degrades | Infinite |
| Agency | Erodes | Preserved |
