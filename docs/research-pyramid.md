## The Intake Void

New sources enter through a **triage grid**:

```
┌────────────────────────────────────────────────────────────┐
│                   SOURCE INTAKE GRID                       │
├──────────────┬──────────────┬──────────────┬──────────────┤
│   EVIDENCE   │   CLAIM      │   MECHANISM  │   OUTCOME    │
│   (What?)    │   (Why?)     │   (How?)     │   (So what?) │
├──────────────┼──────────────┼──────────────┼──────────────┤
│ Archaeological│ Thesis      │ Operator     │ Case Study   │
│ Level 5      │ Level 1     │ Level 2      │ Level 4      │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

For each new source, ask:
1. **What is the evidence?** → Archaeological (base)
2. **What is the claim?** → Thesis (apex)
3. **How does it work?** → Operator (mechanism)
4. **What happened?** → Case Study (outcome)

Sources may span multiple levels — that's fine. The grid reveals where they contribute.
